RTS Agent Generator

RTS Agent Generator is a Unity-based tool developed as part of my thesis, aimed at designing agents for Real-Time Strategy (RTS) games. The system focuses on implementing Steering Behaviors and includes both a custom editor and an in-engine editor for agent configuration. It allows the creation of different types of agents, including resource gatherers, seekers, leaders (using the “Follow the Leader” behavior), and followers. The tool also features a level creation and saving system, enabling users to build and persist custom RTS scenarios for testing or gameplay purposes. This provides a flexible way to prototype and test group-based agent behaviors in a modular and reusable workflow.
