Lost Lantern

2D platformer inspired by Hollow Knight. Main mechanic is a lantern. Enemy AI uses Behavior Trees. All art by me.

void Update() {
    if (enemy.IsPlayerInRange()) {
        enemy.Chase();
    } else {
        enemy.Patrol();
    }
}
